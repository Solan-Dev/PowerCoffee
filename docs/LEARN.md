# Microsoft Learn references (Power Platform CLI + MCP)

## Built-in MCP server
- Use Power Platform CLI with built-in MCP server
  - https://learn.microsoft.com/en-us/power-platform/developer/howto/use-mcp
- CLI reference for `pac copilot mcp`
  - https://learn.microsoft.com/en-us/power-platform/developer/cli/reference/copilot#pac-copilot-mcp

## Key points (from Learn)
- The PAC CLI includes an integrated MCP server for local development and testing.
- Start MCP server with: `pac copilot mcp --run`.
- Register the MCP server in VS Code using MCP configuration.
- Prerequisite listed: .NET 10+.
